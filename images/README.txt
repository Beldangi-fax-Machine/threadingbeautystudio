THREADING BEAUTY STUDIO - IMAGE FOLDERS
=======================================

This images folder contains subfolders for organizing your website images:

/images/reviews/
- Place customer review photos here
- Recommended size: 400x400 pixels or larger
- Supported formats: JPG, PNG, WebP

/images/gallery/
- Place before/after photos and work samples here
- Recommended size: 600x600 pixels (square) or larger
- Categories: eyebrows, full-face, reviews
- Supported formats: JPG, PNG, WebP

/images/team/
- Place team member photos here
- Recommended size: 400x400 pixels (square)
- Supported formats: JPG, PNG, WebP

GENERAL TIPS:
- Use high-quality images for best results
- Optimize images for web (compress without losing quality)
- Use consistent aspect ratios within each category
- Name files descriptively (e.g., "eyebrow-threading-01.jpg")

After adding images, update the HTML files to replace the placeholder divs with actual <img> tags.
